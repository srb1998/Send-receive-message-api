#Conversation API to send and view message Project


## Requirements

1)Install dependencies:

$ npm i express,body-parser,mysql,nodemon

2)Install XAMPP:

-Run http://localhost/phpmyadmin/ and import 'messageinfo' from database folder.

-Setup your MySQL port:3306 in XAMPP Control Panel.


3)Install Hyper Terminal/console

-cd {Project-directory}

-start server - 'nodemon app.js'

4) run 'http://localhost:3000/' and voila! you are good to go.
